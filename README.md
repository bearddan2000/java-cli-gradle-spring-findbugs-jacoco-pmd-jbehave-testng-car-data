# java-cli-gradle-spring-findbugs-jacoco-pmd-jbehave-testng-car-data

## Description
A POC for a spring app using testng, findbugs,
jacoco, pmd, jbehave framework.

## Tech stack
- java
- gradle
	- spring
  - testng  
  - jbehave
  - jacoco
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
